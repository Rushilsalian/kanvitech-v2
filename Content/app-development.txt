FAQ  (also supports SEO)
----------------------------------------------------------
Q: What's the difference between a website and a web application?
A: A website mainly shows information. A web application is interactive - it lets users log in, store data, and do things like online checkout or account management.

Q: Will my site or app work on mobile?
A: Yes. We build everything responsive by default, so it adapts cleanly to phones, tablets and desktops.

Q: Do you build mobile apps too?
A: Yes - alongside websites and web apps, we develop mobile applications depending on what your project needs.

Q: Who owns the code once it's built?
A: You do. The source code, design and rights are yours once the project is complete and final payment is cleared.

Q: How long does a project take?
A: A standard website is usually a few weeks; a custom web or mobile application takes longer. We give you a clear timeline once we know the scope.

Q: What happens after the site goes live?
A: We offer ongoing maintenance - bug fixes, security updates, backups and performance checks - so it keeps running well.
